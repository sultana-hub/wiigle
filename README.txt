To clone from GitHub just type in vs code terminal :navigate to the desired folder and just type the command git clone <project GitHub url>

To upload in github using git bash:

Step 1: Right click on folder-> git bash here
or 
open project in vs code, choose git bash as terminal
 
Step 2: git init  

Step 3: git add . 
 
Step 4: git commit -m "msg" 

[ If 1st time using git in the machine: 
git config --global user.email "your email"  
git config --global user.name "user name" ]
 
Step 5: git branch -M main 

Step 6:  git remote add origin "https://github.com/sultana-hub/wiigle"

Step 7: git push -u origin main   
[1st time user is needed to authorize from browser]
